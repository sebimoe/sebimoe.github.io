Limited Space

Limited Space is a game made in 72 hours for Ludum Dare 54 game jam with the theme "limited space". Developed by me and Luksor.

See entry on ludum dare or play the game in browser

Game objective

Your objective is to type as many letters as possible, by using any words from the English dictionary, using the least possible spaces. Don't make mistakes and don't use banned letters to avoid negative points.

Technology

This game is developed entirely using React and Web technologies. While not using a game engine comes at a cost, in this case the game idea and design were well suited for browser-native implementation.
