Chuuni - Keyboard Game Controller

Chuuni is a game controller for rhythm games. Chuuni is in development.

Chuuni has 64 line-of-sight touch sensors - fast, and unaffected by gloves. Custom arrangement of 64 configurable sensors makes it compatible with common game layouts of 4-16 and more keys.

Features

  • Line-of-sight sensors mean that all touches are registered right when they should, even if you wear boxing gloves.
  • 64 sensors allow you to configure the controller to almost any game.
  • Sensor grouping will allow you to create smaller or larger buttons.
  • Fast response for maximum gameplay performance, unaffected by gloves.

Journey

PCB Design

Chuuni is powered by a set of custom PCBs. The PCBs handle 64 IR sensors and the logic to sense your touches.
